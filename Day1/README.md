# Python Practice Exercises

This repository contains beginner-level Python exercises focused on core programming concepts like variables, functions, loops, and input handling.

---

## Exercise 1: Basic Variables & Calculations

### Objective

Learn how to:

* Declare variables
* Use formatted strings
* Perform simple calculations

### Requirements

* Create variables for:

  * Name (string)
  * Age (integer)
  * Coffee drinking habit (boolean)
  * Salary (float)
* Print a formatted sentence using all variables
* Calculate:

  * Years left until retirement (age 60)
  * Weekly coffee budget:

    * 3 cups/day
    * Rs. 150 per cup

### Concepts Covered

* Variables
* Data types
* Arithmetic operations
* f-strings
* Conditional logic

## Exercise 2: Grade Classifier Function

### Objective

Understand functions and conditional logic.

### Requirements

* Create a function:

  ```python
  grade_classifier(score)
  ```
* Return:

  * `"Distinction"` → score ≥ 90
  * `"Pass"` → 60–89
  * `"Fail"` → < 60

### Tasks

* Test the function with at least 5 values
* Use a loop with:

  ```python
  scores = [45, 72, 91, 60, 38, 85]
  ```
* Print result for each score

### Important

* Do NOT repeat the if-else block multiple times
* Use the function inside the loop

### Concepts Covered

* Functions
* if-elif-else
* Loops
* Code reusability

## Exercise 3: Multiplication Table Generator

### Objective

Work with user input, loops, and formatting.

### Requirements

1. Ask user for a number (1–12)
2. Validate input using a `while` loop

   * If invalid → ask again
3. Print multiplication table for that number
4. Format output neatly (right-aligned)

### Bonus Task

* Print tables for ALL numbers from 1 to 12

### Example Output

```
 5 x  1 =   5
 5 x  2 =  10
 ...
 5 x 12 =  60
```

### Concepts Covered

* User input
* while loop (validation)
* for loop
* Nested loops
* Output formatting
