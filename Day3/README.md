# Task Tracker CLI (Python)

This is a **Command-Line Task Tracker** built in **Python** using **Object-Oriented Programming (OOP)** and **JSON file persistence**.  
It allows you to **add, list, complete, and delete tasks**, and saves all tasks in a `tasks.json` file so that data persists between program runs.

---

## Features

- Add tasks with a title  
- List all tasks or filter by status (`todo` / `done`)  
- Mark tasks as done  
- Delete tasks  
- Tasks persist in `tasks.json`  
- Handles errors like invalid task ID, missing JSON file, or corrupt JSON
